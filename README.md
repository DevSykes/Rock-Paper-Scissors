# 九귏릤넖잺九껊잺 Rock-Paper-Scissors Game 游꿡

Welcome to my personal project. This game is built to practice and improve my **Python** skills.

---

## How to Play

1. Choose your move: Rock, Paper, or Scissors.
2. The computer will choose its move.
3. See who wins.
4. The game keeps track of your wins, losses, and ties with a score counter

Have fun!

---

## Running the Game

To start the game, run the following command in your terminal:

```bash
python3 rock_paper_scissors.py
```
